# HireMyIdea Assignment

An Android app built with Jetpack Compose that displays a health insights dashboard.


## Demo Video Link: https://drive.google.com/file/d/1GCs4_eJScZ0rPJZvhSdtnKtB-j1iodO4/view?usp=sharing


## Features

- **Insights Screen** with scrollable cards:
  - Stability Summary — area chart showing cycle stability score over time
  - Cycle Trends — segmented pill bar chart showing monthly cycle phases
  - Body Signals — donut chart showing symptom breakdown
  - Body Metabolic & Lifestyle Impact cards
- **Custom Bottom Navigation** — floating pill nav bar with Home, Track, Insights tabs and a "+" action button

## Tech Stack

- Kotlin
- Jetpack Compose (Material 3)
- Canvas for all custom charts and icons

## Getting Started

1. Clone the repo
2. Open in Android Studio
3. Run on an emulator or physical device (API 26+)
